## 额外工具箱 

#### 1.动力学物体全局禁用启用  
当场景里有一堆布料软体，播放就会无比的卡，逐个关太麻烦了，所以做了个一键关闭  
先生成动力学物体的关联集合，然后根据这个进行启用禁用  
因为是关联，所以删掉集合也没事  

#### 2.贴图合并(重映射)
按钮位置 - 图像编辑器 - 图片 - mi2bl工具  
两个存档的模型用了很多相同的贴图  
然后追加到另一个存档里，就会重复加载图片生成图片对象，会多出额外的贴图内存  
所以这里先选择要合并的图片，便会遍历检查是否有重复路径的(正则.00x也可以，但不安全)，生成面板以确定删除
